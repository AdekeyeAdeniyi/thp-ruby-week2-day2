# Ruby Tutorial - Week 2 Day 2

## About

This repository contains practice exercises from **The Hacking Project** - Week 2, Day 2 Ruby programming exercises.

## Project Structure

This project includes several Ruby exercises designed to practice fundamental Ruby concepts:

- `exo_01.rb` - Hello World program
- `exo_02.rb` - User input and string interpolation
- `exo_03.rb` - Age calculator with input validation

*More exercises will be added as the project progresses.*

## Exercises Description

### Exercise 1 (exo_01.rb)
Basic "Hello World" program that outputs a greeting message.

### Exercise 2 (exo_02.rb)
Interactive program that:
- Prompts the user for their name
- Greets the user with a personalized message using string interpolation

### Exercise 3 (exo_03.rb)
Age calculator program that:
- Asks for the user's birth year
- Validates that the input is a valid number
- Checks if the birth year is in the future
- Calculates and displays the user's current age

## Requirements

- Ruby (version 2.0 or higher recommended)

## Installation

1. Clone this repository:
```bash
git clone https://github.com/AdekeyeAdeniyi/thp-ruby-week2-day2.git
cd thp-ruby-week2-day2
```

2. Ensure Ruby is installed:
```bash
ruby --version
```

## Usage

Run any exercise file using the Ruby interpreter:
```bash
ruby exo_<number>.rb
```

## Learning Objectives

These exercises cover:
- Basic Ruby syntax
- Output with `puts` and `print`
- User input with `gets.chomp`
- String interpolation
- Type conversion
- Input validation with regular expressions
- Control flow (if/else, while loops)
- Working with the Time class
